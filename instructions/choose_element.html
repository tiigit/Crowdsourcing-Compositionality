How to perform the task:

  1. Read the task instructions below and make sure you understand them before starting the task.
  2. Answer the question: Which diagram element belongs together with the diagram element (or elements) highlighted in red?
  3. Use the point tool to choose the element which, in your opinion, belongs together with the element (or elements) highlighted in red.
  4. Click "Submit" to submit the completed tasks.

Payment information: You will be paid $0.2 per task. You will be paid after the task is completed and submitted.

Purpose statement: These tasks are part of a research project conducted at the University of Helsinki.

Instructions:

In the task you will see a science-themed diagram (see example pictures below).

The diagram contains different types of elements, which can be for example text, arrows, illustrations, colors etc.
In the diagram, one or more elements are highlighted in red. Other elements are highlighted in green.

Your job is to decide which diagram element highlighted in green belongs together with the diagram element (or elements) highlighted in red.
"Belonging together" means that the elements are connected to each other in a sense that together they form some kind of a meaningful pair or group.
  1. Use the zoom tool to make sure you can see the whole diagram.
  2. Find the diagram element (or multiple elements) highlighted in red.
  3. Think about the following:
    Which other diagram element (highlighted in green) belongs together with the element (or elements) highlighted in red?
  4. Choose the green element by placing a point (dot) on top of or inside the element you wish to choose.
