How to perform the task:

  1. Read the task instructions below and make sure you understand them before starting the task.
  2. In the task, you will see an image of a science-themed diagram.
  3. In the diagram, one or more diagram elements will be highlighted.
  4. Find the highlighted diagram element (or multiple elements). Use the zoom tool to make sure you can see the whole diagram.
  5. Answer the question: Which diagram element belongs together with the highlighted diagram element (or elements)?
  6. Use the point tool (shortcut key: ??) to choose the element which, in your opinion, belongs together with the highlighted element/elements.
  7. Click Submit to submit the completed tasks.

Payment information: You will be paid $0.15 per task. You will be paid after the task is completed.

Purpose statement: These tasks are part of a research project conducted at the University of Helsinki.

Instructions

You will see a diagram.
The diagram contains of different types of elements, which can be for example text, arrows, illustrations, colors etc.
Your job is to decide which other element in the diagram belongs together with the highlighted diagram element/elements.
"Belonging together" means that the elements are connected in a sense that they form together some kind of a meaningful pair or a group.

Find the highlighted diagram element/elements and think about the following:
Which element in the diagram belongs together with the highlighted element (or elements)?

Choose the element by placing a point (dot) on top of or inside the chosen element.
